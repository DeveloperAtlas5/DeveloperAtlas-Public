# Open HTML in a Browser

Sometimes `index.html` opens in VS Code instead of your browser.

That is normal.

## On Windows

1. Find `mini-project/index.html`.
2. Right-click the file.
3. Choose **Open with**.
4. Pick Chrome, Edge, Firefox, or another browser.

## What should happen

You should see a small web page with:

```txt
My First Web Page
Click me
```

## If you only see code

You opened the file in an editor.

That is okay, but for testing the page you need the browser.
