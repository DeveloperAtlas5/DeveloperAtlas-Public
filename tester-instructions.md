# Tester Instructions

Thank you for testing Developer Atlas.

## What to do

1. Download the pack you were given.
2. Extract it.
3. Open `mission-dashboard.html` in your browser.
4. Follow only the first visible instruction.
5. Stop when something feels unclear.

## Important

You are not being tested.

The pack is being tested.

If something is confusing, that is useful feedback.

## What to send back

Please answer:

1. What did you open first?
2. What did you understand immediately?
3. Where did you slow down or get confused?
4. Did you know what to do next?
5. What would make it easier?

## For Rookie Pack testers

The Rookie Pack assumes you already have a basic Vue to-do project.

If you do not have a Vue project yet, say that clearly in your feedback.
