# Example — Human Preference Code Review

## User prompt

```txt
Review this code using the AI Human Preference & Readability Standard.
Do not rewrite it yet.
Tell me what is unclear and what the smallest improvement would be.
```

## Good AI behavior

The AI should:

- explain readability issues
- avoid a full rewrite
- suggest the smallest useful change
- preserve the learner's control
- park unrelated improvements

## Bad AI behavior

The AI should not:

- rewrite the entire file immediately
- introduce new architecture
- rename everything without asking
- add unrelated features
- skip the explanation
