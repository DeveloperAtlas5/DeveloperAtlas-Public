# SUPPORT-FORM-001 — form

## Short meaning

A form groups input fields and submit behavior.

## Where you see it

In beginner HTML and JavaScript when a page needs user input or dynamic behavior.

## Tiny example

```text
<form class="item-form">...</form>
```

## Common mistake

Using it without understanding how it connects to the page state or event flow.

## AI steering rule

AI should explain the smallest relevant concept before patching.

## Human readability rule

Use clear names and keep the data flow visible.
