# Lesson 4 — Connect the Files

The HTML file connects to the CSS and JavaScript files.

Open:

```txt
mini-project/index.html
```

Look for:

```html
<link rel="stylesheet" href="style.css">
```

This connects CSS.

Look for:

```html
<script src="script.js"></script>
```

This connects JavaScript.

## Why this matters

If a file name is wrong, the browser may not load the style or behavior.

## Tiny check

Make sure these files are in the same folder:

```txt
mini-project/
  index.html
  style.css
  script.js
```
