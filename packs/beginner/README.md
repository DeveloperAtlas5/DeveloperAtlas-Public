# Developer Atlas Beginner Preview

## Best for

Use this preview if:

- you are new to web projects
- you do not know what HTML, CSS, and JavaScript each do
- you are not ready for the Vue Rookie Pack yet
- you want a tiny first win before using a larger project

## What you will test

You will test whether the first beginner instructions feel clear enough to continue.

This preview teaches:

```txt
HTML = page structure
CSS = page style
JavaScript = page behavior
```

## Expected time

10–20 minutes.

## Start here

Open:

```txt
START_HERE.md
```

## What you need

You only need:

- a browser, like Chrome, Edge, or Firefox
- a text editor, like VS Code or Notepad

You do not need Node, npm, Vue, Laravel, or a terminal for this preview.

## Important

You do not need to understand everything.

The goal is to know what to open, what to change, and how to see the result.
