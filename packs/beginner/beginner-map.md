# Beginner Map

A tiny web project often has three files.

```txt
index.html
style.css
script.js
```

## HTML

HTML is the structure.

It answers:

```txt
What is on the page?
```

Examples:

```txt
heading
paragraph
button
input
image
link
```

## CSS

CSS is the style.

It answers:

```txt
How does the page look?
```

Examples:

```txt
color
spacing
size
layout
border
background
```

## JavaScript

JavaScript is the behavior.

It answers:

```txt
What happens when the user does something?
```

Examples:

```txt
click a button
change text
show a message
save simple data
```

## Simple memory rule

```txt
HTML = what exists
CSS = how it looks
JavaScript = what happens
```
