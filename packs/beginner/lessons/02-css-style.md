# Lesson 2 — CSS Style

CSS controls how the page looks.

Open:

```txt
mini-project/style.css
```

Look for:

```css
body {
  font-family: Arial, sans-serif;
  max-width: 720px;
  margin: 40px auto;
  padding: 24px;
}
```

## What this means

```txt
font-family = text style
max-width = how wide the page can get
margin = space outside
padding = space inside
```

## Tiny task

Find this line:

```css
border-radius: 8px;
```

Change `8px` to `16px`.

Save and refresh the browser.

The box corners should look more rounded.
