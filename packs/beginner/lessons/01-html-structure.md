# Lesson 1 — HTML Structure

HTML tells the browser what exists on the page.

Open:

```txt
mini-project/index.html
```

Look for:

```html
<h1>My First Web Page</h1>
<p>This page has structure, style, and behavior.</p>
<button id="messageButton">Click me</button>
```

## What this means

```txt
<h1> = big heading
<p> = paragraph text
<button> = clickable button
```

## Tiny task

Change:

```txt
My First Web Page
```

to:

```txt
My First Developer Atlas Page
```

Save and refresh the browser.
